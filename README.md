# Staff Teacher App

A comprehensive Flutter mobile application designed to simplify and digitize the daily operations of school staff and teachers. The app provides a centralized platform for managing academic, administrative, and communication tasks efficiently.

## Features

This application encompasses a wide variety of features tailored for teachers and school staff:

### 🎓 Academics & Classroom Management
- **Class & Sections:** Manage and view assigned classes and sections.
- **Student Profiles:** View detailed profiles and performance metrics of students.
- **Timetable:** Access daily and weekly class schedules.
- **Subjects & Lessons:** Track subject progress and structure lesson plans.
- **Attendance:** Easily record and manage daily student attendance.
- **Assignments & Exams:** Create, assign, and track student assignments and examination details.
- **Diary:** Maintain a digital diary for daily class activities.

### 🏢 Administration & HR
- **User Profiles:** Secure login, change passwords, and edit personal teacher profiles.
- **Leaves Management:** Apply for leaves, manage requests, and track leave status.
- **Holidays:** View the academic calendar and upcoming public or school holidays.
- **Payroll:** Access payroll and salary information.
- **Transportation:** View school transport routes and relevant schedules.

### 💬 Communication & Notifications
- **Chat System:** Integrated messaging for staff-to-staff communication.
- **Notifications:** Real-time push alerts and a dedicated notification center for important updates and school announcements.
- **Contacts:** Institutional directory of essential contacts.

## Technical Details

- **Framework:** [Flutter](https://flutter.dev/)
- **Language:** Dart
- **SDK Version:** `^3.10.7`
- **Key Dependencies:**
  - `cupertino_icons`: For iOS-style icons.
  - `google_fonts`: For modern and responsive typography.
- **UI Architecture:** The app UI is highly modularized under the `lib/screens/` directory, separating concerns into dedicated files like `home_screen.dart`, `login_screen.dart`, `attendance_screens.dart`, etc.

## Getting Started

To explore and run this project locally, ensure you have the Flutter SDK installed on your machine.

1. Navigate to the project directory:
   ```bash
   cd "d:\Mobile Application\stuff_teacher"
   ```
2. Install dependencies:
   ```bash
   flutter pub get
   ```
3. Connect your Android/iOS device or start an emulator.
4. Run the app:
   ```bash
   flutter run
   ```
